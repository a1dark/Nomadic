# Nomadic

## 🌾 About

**Nomadic** is a work-in-progress Minecraft mod created by **a1dark**.

The goal of this project is to introduce nomadic cultural elements inspired by Central Asian nomadic traditions into Minecraft.

This is my first mod project and is mainly developed for learning Java modding and game development.

##  Vision

- Add nomadic-themed content to Minecraft
- Introduce culturally inspired gameplay elements
- Experiment with Forge mod development
- Represent Central Asian nomadic heritage in a game environment

 The project is currently under active development.

##  Development Status

- 🚧 Early development
- 🐛 Bugs and instability are expected
- 🧠 Architecture is still being improved

## Technology Stack

- Java **21**
- Minecraft **Forge**
- Minecraft Version: **Forge 1.21.1**

## 🛠 Requirements

To build this mod you need:

- Java 21+
- Gradle
- Compatible Minecraft Forge 1.21.1 setup
